# 联系我入口与联系方式展示

## 背景

用户希望 Codex 注入弹窗和管理工具关于页都展示可联系渠道，方便用户加入官方 QQ 群或通过微信二维码进行合作代理联系。当前注入弹窗只有主页、推荐内容、支持；关于页也只展示版本、项目地址和 Release 更新。

## 目标

本次要完成：
- 在 Codex 注入弹窗 tab 区新增“联系我”入口。
- 点击“联系我”后展示官方 QQ 群一键添加入口和合作代理微信二维码。
- 在管理工具“关于”页面新增同样的联系方式展示。
- 使用用户提供的微信二维码图片作为本地项目资产。

本次不包含：
- 不修改 Claude 中文注入。
- 不修改盘古记忆、插件仓库、供应商配置。
- 不改变现有主页、推荐内容、支持页功能。
- 不新增后端接口。

## 用户视角描述

用户点击 Codex 顶部注入标识，打开功能弹窗后，可以点击“联系我”tab。页面第一行展示两个官方 QQ 群号和“一键添加”链接；第二行展示“合作代理请联系微信”和微信二维码。用户在管理工具关于页也能看到同样信息。
