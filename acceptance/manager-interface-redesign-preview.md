# 验收标准：管理工具界面 UI 重设计评审原型

验证对象：`spec/manager-interface-redesign-preview.md`

## 验收项

1. 文档存在
   - 通过标准：规格与验收文档存在。
   - 证据：文件存在检查。

2. 预览文件存在
   - 通过标准：`docs/ui-previews/manager-interface-redesign/index.html` 存在。
   - 证据：文件存在检查。

3. 覆盖现有页面
   - 通过标准：预览包含概览、供应商、工具与插件、会话管理、维护、设置、关于 7 个页面入口和页面内容。
   - 证据：源码检查与截图。

4. 不应用到真实前端
   - 通过标准：本任务不修改真实 React/Tauri 页面逻辑，不写入 `apps/claude-codex-pro-manager/src/App.tsx`。
   - 证据：`git status` 与源码检查。

5. 不调用真实接口
   - 通过标准：预览不包含 `fetch(`、`invoke(` 或 `__TAURI__`。
   - 证据：静态源码检查。

6. UI 风格符合要求
   - 通过标准：呈现低噪声深色工程控制台；表面层级、状态标签、列表、表单、按钮和开关风格统一。
   - 证据：截图与人工检查。

7. 不恢复已删除设置项
   - 通过标准：预览设置页不出现 `Codex 启动参数`、`图片覆盖`、`安全边界`、`用户脚本` 等旧卡片。
   - 证据：源码关键词检查。

8. 工具与插件开关对齐
   - 通过标准：开关胶囊与圆球上下居中，条目右侧操作槽稳定。
   - 证据：截图或源码检查。

9. 基础交互可用
   - 通过标准：点击导航可切换页面；预览开关可切换视觉状态。
   - 证据：浏览器截图或手动检查。

## 必需验证

至少执行：

```powershell
Test-Path docs\ui-previews\manager-interface-redesign\index.html
```

并使用浏览器或等效工具打开预览，生成或检查 7 个页面截图。

## 不在范围内

- 真实前端落地。
- 后端命令调用验证。
- Tauri debug/release 构建。
- 发布安装包。
