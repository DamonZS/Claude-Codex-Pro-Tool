# 盘古记忆成果看板验收标准

对应规格：`spec/pangu-memory-outcome-dashboard.md`

## 通过/失败标准

1. **真实事件**：搜索命中、会话注入、MCP 搜索分别落库并带正确来源；普通列表不产生召回。任一来源缺失则失败。
2. **非阻断**：模拟活动事件写入失败时，主查询仍返回命中结果。事件失败导致查询失败则失败。
3. **脱敏与隔离**：查询中的 `api_key`、`token`、`access_token`、`password`、`secret`、Bearer/Basic 凭据被脱敏且摘要受长度限制；命中 global 记忆的事件仍按查询来源 workspace 隔离，workspace A/global 看板不返回 workspace B 查询摘要。泄漏则失败。
4. **真实聚合**：7/30 天趋势、今日采集/新增/召回、项目和类别分布均由测试数据库记录计算，结果与夹具一致。
5. **可审计证据**：最近召回包含时间、Agent、方式、查询摘要与命中时的脱敏限长记忆快照；记忆后来编辑、移动后仍展示原快照而非当前正文；无数据时 UI 明确写“尚无可验证召回记录”。
6. **项目接续**：当前 workspace 展示关键规则/修复/进度等真实记忆，并可复制给 Agent。
7. **简化默认页**：首屏以项目接续、成果、趋势、召回证据、待确认、项目记忆为主；三层链路、采集、手册、遗忘、MCP、导入导出位于默认闭合的高级诊断区。
8. **谨慎文案**：页面不出现“系统证明避免了错误”等结论，仅陈述真实召回或注入。
9. **兼容性**：现有记忆列表、编辑、归档/恢复、候选审批、MCP 工具和注入流程仍可用。
10. **范围控制**：无新 UI 依赖，不改其他导航/页面，不修改火焰特效，不删除用户数据。

## 必需验证

- `cargo test -p claude-codex-pro-core --manifest-path Cargo.toml memory_assist -- --nocapture`
- MCP 的定向测试，证明 `memory_search` 来源事件。
- `npm --prefix apps/claude-codex-pro-manager run check`
- `npm --prefix apps/claude-codex-pro-manager run vite:build`
- `cargo test -p claude-codex-pro-manager --manifest-path Cargo.toml --test windows_subsystem -- --nocapture`
- 本地打开盘古记忆页，截图证明默认信息架构与高级诊断闭合状态；若运行环境无法启动，必须明确记录未验证风险。

## 完成证据

- 测试输出显示事件记录、workspace 隔离、脱敏、聚合和非阻断行为通过。
- 前端类型检查、构建及 Windows 契约测试通过。
- 页面截图不包含 API key、完整 token、敏感 Base URL 或用户私密会话全文。

## 非目标检查

- 不要求知识图谱可视化。
- 不要求自动判断某次召回确实阻止了错误。
- 不要求将活动查询历史加入用户记忆导出。

## 新项目启动指南验收

11. **跨项目精选**：global 与多个 workspace 的合格活跃经验参与汇总，归档项不参与；结果不暴露 workspace 名称或会话 ID。
    - global 经验允许独立入选；单个私有 workspace 独有经验不得进入指南，非 global 经验必须由至少两个不同私有 workspace 的相似经验共同印证。
12. **泛化与安全**：绝对/相对脚本路径、通用 Token/Key/Password/Secret、Bearer/Basic 凭据、敏感 URL、全库任一 workspace/会话标识、项目专属命令和无法泛化内容不进入指南；相近经验被去重并保留来源数量。
13. **确定性与限额**：相同数据库重复生成结果一致，避坑和优秀做法各不超过 12 条；无合格经验返回真实空态。
14. **召回边界**：生成、预览或复制新项目指南不增加搜索/注入召回计数。
15. **提示词内容**：完整提示词包含读文档与源码、实施前总结、最小修改、可复现验证、真实测试证据和按新项目规则调整六类固定要求。
16. **双场景交互**：首屏同时提供继续当前项目与开启新项目；新项目必须先预览再复制，两类复制内容互不混入项目私有细节。
17. **易用化页面**：默认主要区域为开始工作、今日成果、趋势/分布、召回/待确认；项目记忆紧凑可展开，高级诊断默认关闭，窄屏布局为单列。

新增验证证据：Core 新项目指南筛选测试、Tauri/前端类型契约、双场景页面截图及预览复制手动检查。

补充契约检查：指南返回稳定的生成时间、唯一源记忆数量和来源项目数量；完整提示词实际包含筛选出的避坑经验与优秀做法，且统计字段不泄露 workspace 名称或会话标识。
